# Neil Life Foundation Website

Official Next.js website for **Neil Life Foundation**.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

If you ever see stale chunk errors like `Cannot find module './611.js'`, run a fresh start:

```bash
npm run dev:fresh
```

## Scripts

- `npm run dev` - Start development server
- `npm run clean` - Remove local build/cache logs safely
- `npm run dev:fresh` - Clean and start development server
- `npm run build` - Build for production
- `npm run build:fresh` - Clean and build for production
- `npm run start` - Run production build
- `npm run lint` - Run lint checks
